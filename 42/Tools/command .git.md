## Commandes GIT

##### Créer un repo à partir du clé SSH 
```
git clone git@github.com:MorganeLin/Obsidian.git nomDuRepo
git add . 
git commit "push"
git push
```

##### Pour changer le message du commit
```
git commit --ammend
```
##### Pour changer de depôt 
```
git remote set-url origin <nouvelle_url>
```
