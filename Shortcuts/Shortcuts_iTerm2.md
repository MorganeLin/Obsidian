**Pour clear le terminal :**

	Ctrl + D

**Par rapport aux tabs :** 

	New tab : Cmd + t 
	Close tab : Cmd + w 
	Next tab : Cmd + right
	Previous tab : Cmd + left

**Split screen :** 

	Split Vertically : Cmd + d 
	Split Horzontaly : Cmd + shift + d

**Et afin de move à travers les différents fenêtres splits** 

	Cmd + Opt + left/right/top/bottom
