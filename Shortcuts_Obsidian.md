## Shortcuts Obsidian

##### Comment naviguer

	Cmd + <- / ->  -  Pour naviguer entre les pages
	Cmd + f  -  Faire une recherche 
	Cmd + t  -  Pour ouvir un nouveau tab
	Cmd + w  -  Pour fermer le tab
	Cmd + e  -  Mode edit

##### Editer son texte

	Cmd + b  -  Pour mettre en gras le mot
	Cmd + i  -  Pour mettre un mot en italique
	Cmd + l  -  Pour faire une case à cocher
	~ - Cela barre le texte
	== - Pour mettre en évidence le(s) mot(s)
	un tiret - création d'un rond pour faire une liste
	1 - Numéroter (+ un .) 
	--- - création d'une ligne continu
	Une seule quote - Pour surligner en gris
	Triple quote - Création d'un bloc
     > - Création d'une barre sur le coté/ citation

##### Création de lien 

	- Faire des doubles crochets afin qu'on puisse cliquer dessus en créer un fichier
	- [ ] (le lien) - Création d'un via un alias clickable
